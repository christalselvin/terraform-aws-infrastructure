# Terraform AWS Infrastructure

This project demonstrates Infrastructure as Code using Terraform to provision AWS infrastructure.

Terraform automates the creation of an EC2 instance using configuration files.

---

## Technologies Used

- Terraform
- AWS
- Infrastructure as Code
- DevOps

## Terraform Workflow

Terraform → AWS Provider → Create EC2 Instance → Infrastructure Deployment

---

## Commands Used

Initialize Terraform

terraform init

Validate configuration

terraform validate

View infrastructure plan

terraform plan

Apply infrastructure

terraform apply

Destroy infrastructure

terraform destroy
